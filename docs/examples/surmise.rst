persistent_surmise
------------------

Required: Surmise_

Surmise can be pip installed::

    pip install surmise

The :doc:`Borehole Calibration tutorial<../tutorials/calib_cancel_tutorial>` uses this generator as an example of the capability to cancel pending simulations.
